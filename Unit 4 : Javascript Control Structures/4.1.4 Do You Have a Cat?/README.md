# 4.1.4: Have a Cat - Code & Documentation

## 📋 Copy Code Below
```javascript
function start() {
    // Declare a boolean variable - change true/false depending on whether you have a cat
    var haveCat = true;   // ← Change to false if you don't have a cat
    
    // Print the value of the boolean
    println(haveCat);
}
```

---

## 📖 Project Overview
This exercise introduces the **Boolean** data type in JavaScript. Booleans are the simplest form of data, representing a binary choice: either `true` or `false`.

### How it Works
1. **Initialization**: We use `var` to create a space in memory called `haveCat`.
2. **Assignment**: We store the value `true` inside that variable.
3. **Output**: We use `println()` to send that value to the screen.

### Logic Summary
| State | Boolean Value |
| :--- | :--- |
| I have a cat | `true` |
| I do not have a cat | `false` |
