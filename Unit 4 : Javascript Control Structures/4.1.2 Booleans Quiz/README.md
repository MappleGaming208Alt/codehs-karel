# 4.1.2 Booleans Quiz

## Question 1
**Which of the following is *not* a valid value for a boolean?**

* [ ] `true`
* [ ] `false`
* [x] `yes`
* [ ] The above are all valid

---

### Explanation
In programming, a **Boolean** is a data type that has only two possible values: **true** or **false**. 

While a human might understand "yes" or "no" as being similar to true or false, computers require the specific keywords `true` and `false`. Using `yes` would typically result in a reference error or be treated as a string, but it is not a valid Boolean literal.
